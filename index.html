Welcome to SM1AC

SM1AC is an open-source research project focusing on QEMU-based virtualization and device-model experiments for Apple Silicon-era systems.

Steps to get started

  1. Fork the repository on GitHub.
  2. Clone it to your local machine and read the README.
  3. Run the base QEMU boot script (Linux ARM64) and report results.
  4. Open issues or submit pull requests with reproducible changes.
